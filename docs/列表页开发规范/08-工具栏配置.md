# 工具栏配置

## 使用 DataTableToolbar

```tsx
import { DataTableToolbar } from '@/components/data-table'

<DataTableToolbar
  table={table}
  searchPlaceholder='搜索...'
  searchKey='name'  // 指定搜索字段，不指定则全局搜索
  filters={[
    {
      columnId: 'status',
      title: '状态',
      options: [
        { label: '启用', value: 'active' },
        { label: '禁用', value: 'inactive' },
      ],
    },
    {
      columnId: 'type',
      title: '类型',
      options: [
        { label: '类型A', value: 'a' },
        { label: '类型B', value: 'b' },
      ],
    },
  ]}
/>
```

---

## 工具栏功能

| 功能 | 说明 |
|------|------|
| 搜索框 | 支持指定字段或全局搜索 |
| 多选筛选器 | FacetedFilter，支持多选 |
| 列显隐控制 | ViewOptions，控制列的显示/隐藏 |
| 重置按钮 | 有筛选条件时自动出现 |

---

## Props 说明

| 属性 | 类型 | 说明 |
|------|------|------|
| `table` | `Table<TData>` | useReactTable 返回的实例 |
| `searchPlaceholder` | `string` | 搜索框占位符 |
| `searchKey` | `string` | 搜索字段，不指定则全局搜索 |
| `filters` | `FilterConfig[]` | 筛选器配置数组 |

### FilterConfig 类型

```tsx
interface FilterConfig {
  columnId: string        // 列 ID
  title: string           // 筛选器标题
  options: {
    label: string         // 选项显示文本
    value: string         // 选项值
    icon?: ComponentType  // 可选图标
  }[]
}
```

## 完整示例

```tsx
import { DataTableToolbar } from '@/components/data-table'
import { CircleCheck, CircleX, Clock } from 'lucide-react'

const statusOptions = [
  { label: '启用', value: 'active', icon: CircleCheck },
  { label: '禁用', value: 'inactive', icon: CircleX },
  { label: '待审核', value: 'pending', icon: Clock },
]

const typeOptions = [
  { label: '限时特惠', value: 'flash_sale' },
  { label: '新人专享', value: 'newcomer' },
  { label: '满减活动', value: 'threshold' },
]

<DataTableToolbar
  table={table}
  searchPlaceholder='搜索活动名称...'
  searchKey='name'
  filters={[
    { columnId: 'status', title: '状态', options: statusOptions },
    { columnId: 'type', title: '类型', options: typeOptions },
  ]}
/>
```
