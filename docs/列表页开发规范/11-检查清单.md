# 检查清单

开发新列表页时，请确认以下要点：

## 基础架构

- [ ] 文件结构符合规范（index.tsx + components/{table, columns, dialogs}）
- [ ] 布局使用 `Header` + `Main`，**不使用** Card 包裹表格
- [ ] 弹窗组件已拆分为独立文件，且遵循《弹窗开发规范》

## 表格交互

- [ ] 加载状态使用了 **Skeleton 骨架屏**
- [ ] 服务端分页配置正确 (`manualPagination: true`)
- [ ] 暂无数据时显示了友好的 Empty State
- [ ] 翻页使用 `DataTablePagination`
- [ ] 工具栏使用 `DataTableToolbar`

## 列与内容

- [ ] **操作列**包含：查看(Sheet)、编辑(Dialog)、删除(Confirm)
- [ ] **操作列**已处理 `e.stopPropagation()` 阻止冒泡
- [ ] **删除操作**有红色警示样式及分割线
- [ ] **长文本列**添加了 `truncate` 和 `max-w` 限制
- [ ] **金额/数字列**设置为右对齐 + 等宽字体
- [ ] **日期时间列**经过格式化（非 ISO 字符串）
- [ ] **状态列**使用了标准颜色映射 (`Badge variant='outline'`)

---

## 快速对照表

| 检查项 | 正确做法 | 错误做法 |
|--------|---------|---------|
| 表格容器 | `Main` 组件内 | Card 包裹 |
| 加载状态 | Skeleton 骨架屏 | "加载中..." 文字 |
| 长文本 | `truncate` + `max-w` | 无限制撑开 |
| 数字列 | 右对齐 + `font-mono` | 左对齐 |
| 日期时间 | 格式化显示 | ISO 字符串 |
| 删除操作 | 分割线 + 红色 | 无区分 |
| 弹窗代码 | 独立文件 | 内联在主页面 |
